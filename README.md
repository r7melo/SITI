# SITI

###  Baixar o projeto completo
```
git clone --recursive https://github.com/r7melo/SITI.git
```

### Padrões de commit

- feat:     Uma nova funcionalidade (Ex: feat(api): cria busca de usuarios)
- fix:      Correção de um erro/bug (Ex: fix(web): corrige erro no botao salvar)
- docs:     Mudanças na documentação (Ex: docs: atualiza guia de instalacao)
- style:    Formatação, CSS, UI/UX (Ex: style(web): ajusta cores do dashboard)
- refactor: Melhoria no código sem mudar lógica (Ex: refactor(api): limpa imports)
- chore:    Configurações, dependências, Git (Ex: chore: atualiza .gitmodules)
- test:     Adição ou alteração de testes (Ex: test(api): cria teste de login)
